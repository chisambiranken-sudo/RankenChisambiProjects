# Smart Car Parking System

## Overview
A smart parking system developed using Arduino to automate vehicle counting and parking space monitoring.

## Technologies Used
- Arduino Uno
- Ultrasonic Sensors
- Servo Motor
- Bluetooth Module
- C++

## Features
- Vehicle counting
- Parking availability display
- Automatic gate control
- Bluetooth monitoring
